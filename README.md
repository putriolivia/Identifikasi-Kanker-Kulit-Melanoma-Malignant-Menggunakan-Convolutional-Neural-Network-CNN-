# Identifikasi Kanker Kulit Melanoma Malignant Menggunakan Convolutional Neural Network (CNN)

Proyek ini dibuat sebagai pemenuhan Mata Kuliah Metode Penelitian, dengan fokus pada pembangunan model deep learning untuk mengidentifikasi kanker kulit melanoma berdasarkan citra lesi kulit. Melanoma merupakan salah satu jenis kanker kulit yang berbahaya dan membutuhkan deteksi dini untuk mencegah penyebaran ke organ lain.

Dengan memanfaatkan dataset Melanoma Cancer Image Dataset dari Kaggle, penelitian ini menggunakan algoritma Convolutional Neural Network (CNN) untuk membedakan lesi benign (jinak) dan malignant (ganas). CNN dipilih karena kemampuannya dalam mengekstraksi fitur visual secara otomatis dan akurat pada data citra.

Model CNN yang dibangun menghasilkan akurasi sebesar 88%, menunjukkan bahwa model memiliki performa yang cukup baik dalam membantu proses identifikasi melanoma.
